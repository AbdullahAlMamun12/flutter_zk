## 1.0.0

* Initial Release.
* Added comprehensive documentation to all public members.
* Implemented new features:
    *   Enhanced device information retrieval (Platform, MAC Address, Device Name, Face and Fingerprint Algorithm Versions, Network Parameters).
    *   Full user management capabilities (add, update, delete users).
    *   Attendance record fetching.
    *   Device control operations (enable/disable, restart/power off, refresh data, unlock doors, play voice messages).
    *   Data management functions (clear all data, clear attendance records).
